How this ERP is monitored

This platform runs as a set of Docker containers defined in docker-compose.yml. Alongside the ERP website (this site), three monitoring containers track resource usage of every container in real time — no code changes to the ERP app are needed for this to work.

cAdvisor

Collects per-container CPU, memory, network & disk stats.

Open :8080 →
Prometheus

Scrapes and stores metrics from cAdvisor every 15s.

Open :9090 →
Grafana

Visual dashboards built on top of Prometheus data.

Open :3000 →
